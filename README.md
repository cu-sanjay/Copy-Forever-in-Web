# Clipboard Eternity: Copy Forever

A lightweight browser extension that automatically saves everything you copy.
No setup. No accounts. No cloud. Just copy and it is saved.

## What it does

Clipboard Eternity quietly runs in the background and remembers everything you copy:
- Text
- Links
- Code snippets

You can open the popup anytime and instantly find what you copied before.

## Why it exists

People copy things all day and forget them within seconds.
This extension solves that problem by turning your clipboard into a searchable history.

## Features

- Automatic copy detection (Ctrl + C / right-click copy)
- Instant clipboard history
- Fast search
- One-click copy again
- Smart link detection (URLs are recognized automatically)
- Clean and minimal interface

## Privacy First

Your data stays with you.
- Everything is stored locally in your browser
- No servers
- No tracking
- No analytics
- No data collection

Nothing ever leaves your device.

## How it works

1. You copy something
2. The extension saves it locally
3. You open the popup
4. Search and reuse instantly

## Permissions

- **clipboardRead** => to access copied content after user action  
- **storage** => to save your clipboard history locally  

These permissions are used only for core functionality.

## Installation

1. Download or clone this repository
2. Open Chrome or Edge
3. Go to **chrome://extensions**
4. Enable Developer Mode
5. Click **Load unpacked**
6. Select the project folder
